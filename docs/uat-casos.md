# Matriz UAT — WebPortal/ODP (resumen)

## Categorías principales
- Básicos: startup, login (por rate plan), main framework.
- Home: dashboard consumo, consumo diario, saldo.
- Mi Plan: paquetes suscritos, desuscripción.
- Paquetes: catálogo, suscripción (prepago/pospago/híbrido) por canal de pago.
- Notificaciones: cuota=0, saldo insuficiente, roaming, promoción, birthday, impago.
- ODP (extra): anti-disturb, inbox, theme, tutorial, feedback.

> Referencias de casos y subcasos: usar tablas de aceptación internas (Pass/Fail/NA).
