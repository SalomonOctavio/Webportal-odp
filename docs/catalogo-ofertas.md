# Catálogo de ofertas — estructura (máscara)

## Campos mínimos
- ID oferta, nombre comercial, segmento, vigencia, precio
- Método(s) de pago permitido(s): saldo / ciclo / tarjeta
- Parámetros PCRF: RG, QoS, quota
- Reglas de elegibilidad (plan, saldo mínimo, límites diarios)

## Notas de operación
- Autenticación reforzada cuando se superan umbrales diarios de suscripción.
- Mensajería para rechazo (saldo insuficiente, no elegible).
